# Kubo Search Docs

Repositorio de documentación técnica versionada para la plataforma Search & AI basada en:

- Apache Solr
- Node.js (Backend / API)
- Next.js (Frontend)

---

## Objetivo

Centralizar estándares, arquitectura, schemas y pipelines para:

- estandarizar desarrollo
- escalar la plataforma
- reutilizar en clientes (Kubo Global)
- evitar decisiones inconsistentes
- documentar conocimiento técnico
